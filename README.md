# Development
## Prerequisites
- Yarn

## Installation
### Installing deps
As simple as running yarn.
In the frontend folder run:

```bash
$ yarn install
```

### Building
Can be built in either production mode (minified):

```bash
$ yarn bp
```

Or in development mode:

```bash
$ yarn b
```
